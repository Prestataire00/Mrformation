Généré le 2026-05-25. **4 CHANTIERS DE SOLIDIFICATION MERGÉS EN PROD** (P0 RLS+trigger+enum+scoring `b239757` ; Hygiène B/C/F+multiple_choice `0162ad1` ; Volet D UX pilotage `85aee65` ; P0-5 Auto Qualiopi lien token public `b98f390`). - [Onglet Émargement (TabEmargements + TabAbsences)](./deep-dive-tab-emargements.md) — Analyse complète de l'onglet Émargement (1519 LOC composants + ~4 488 LOC totales lues : 17 routes API émargement/signatures, 4 migrations SQL, 5 pages signature dont `/sign/[token]` publique, services `load-signatures.ts`, composants `src/components/signatures/`).
